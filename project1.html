ToolArchive //June 2017

Description

The ToolArchive was a project within Studio Lust in Den Haag. The projects goal was to create a digital archive of the created tools within the studio. Tools were things such has simple but custom build webcrawlers, control software for museums installations or other custom build software. The intention was to create a plattform where designers and developers could equally access the tools, to either inform themselves over the expertise within the studio or access sourcecode. Therefore, the archive had to be functional but also visually appealing and inspiring for the design team.
